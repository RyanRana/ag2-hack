# Pulse data sources

## Pesticide chemistry — `pesticide_chemistry.json`

Per-chemistry physicochemical properties (DT50, vapor pressure, Henry's law
constant, log Kow, soil Koc, non-target threshold ppm). Sourced from:

- **EPA Pesticide Properties DataBase (PPDB)** — University of Hertfordshire
  AERU, hosted by the US EPA. https://sitem.herts.ac.uk/aeru/ppdb/
- **PAN Pesticide Database** — http://www.pesticideinfo.org/

Listed chemicals: glyphosate, 2,4-D, atrazine, chlorpyrifos.

## Soil-water retention — van Genuchten parameters

Hardcoded per soil-texture class in
`pulse/biophysics/water_balance.py::SOIL_TEXTURES`. Source:

- **USDA-NRCS** soil texture classification — Carsel & Parrish 1988.
  https://www.nrcs.usda.gov/

## Toxicity / LC50 — `pulse/biology/lotka_volterra.py::TOXICITY_DB`

Predator/parasitoid LC50 values per chemical class. Sourced from:

- **EPA ECOTOX** — https://cfpub.epa.gov/ecotox/
- **EPA PPDB** (above)

## Demo imagery — `data/demo/`

Synthetic image (`synthetic_field.jpg`) generated by Pulse for development.
For production demo runs, hand-curate field photos from public Flickr /
Unsplash with clear weed-and-crop scenes, per spec §14.C.
